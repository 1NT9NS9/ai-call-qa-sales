# Stage Completion Criteria

This directory defines completion and handoff criteria for each implementation stage from [../PLAN.md](../PLAN.md).

Each stage document defines:

- objective
- in-scope work
- required deliverables
- exit criteria
- readiness conditions for the next stage
- stage constraints

Stages are sequential. A stage is complete only when all exit criteria in the corresponding file are satisfied.

Stage documents:

- [Stage 0. Setup](stage-0-setup.md)
- [Stage 1. Database](stage-1-database.md)
- [Stage 2. Transcription](stage-2-transcription.md)
- [Stage 3. RAG](stage-3-rag.md)
- [Stage 4. Analysis](stage-4-analysis.md)
- [Stage 5. Export](stage-5-export.md)
